Recruteurs, recruteuses, je vous ai compris!

En ces temps incertains où le monde du travail doit évoluer rapidement, les exigences en matière de compétences techniques et comportementales sont de plus en plus recherchées.

Mais où trouver cette perle rare qui serait allier adaptation et connaissances?

C'est simple, il suffit de prendre sous votre aile un junior plein d'ambitions, de motivations, de volonté et ayant fait preuve de capacités d'adaptation. Aidez-le à se perfectionner, donnez-lui les bons outils et vous finirez par trouver ce que vous cherchiez.

À ce sujet, J'ai effectué un stage de développement web chez Mine of War en télétravail, ma mission étant la mise en place d'un système de gestion du staff dans la partie Admin du site. J'ai travaillé pour la première fois avec Nuxt.js et j'ai eu l'occasion de créer des mocks avec Amazon Web Service.

Suite à ce stage, j'ai profité de l'occasion pour suivre des cours de conduite. Il me reste environ 5h de conduite pour passer mon permis B.

Parallèlement, j'ai cherché à rendre mon temps utile. J'en ai profité pour compléter mes compétences avec Nuxt.js et Node.js pour créer un site complètement fonctionnel et sécurisé qui permet de créer des listes des courses avec du matériel spécifique du bâtiment. J'ai créé ce site dans le but de simplifier la démarche (et donc de gagner du temps) sur la gestion du matériel de chantier.

Mes compétences

AGILE

Les méthodes agiles caractérisent un mode de gestion des projets informatiques privilégiant le dialogue entre toutes les parties prenantes, clients, utilisateurs, développeurs et autres professionnels du projet, la souplesse en cours de réalisation, la capacité à modifier les plans et la rapidité de livraison. Ceci permet d'avoir un résultat au plus proche des attentes du client.

HTML

HTML est un langage courant du Web qui permet d’écrire du texte et d’ajouter des médias (images, vidéos, sons…), il ne permet en revanche aucune couleur ni organisation par exemple.

CSS

Grâce au langage CSS, il est possible de définir des règles appliquées aux documents HTML. Le langage CSS améliore également l’accessibilité des documents, les règles d’affichage selon le support (navigateur, support mobile, télévision, lecteur braille…) ainsi que le style sont également modifiables grâce au CSS. En comparaison avec HTML, CSS ajoute de nouvelles fonctionnalités de style.

JAVASCRIPT

Le langage JavaScript est principalement employé pour améliorer l'ergonomie d'un site Internet et/ou d'une interface applicative utilisateur. Il sert également à intégrer des effets esthétiques.

NODE

Node.js sert à faire du Javascript server side, et peut être utilisé dans des applications de bases de données, la plus populaire étant MySQL. Il dispose également d'une communauté très active. Son principal atout est la possibilité de coder en Javascript, un langage de programmation déjà connu.

REACT

React (ou React.js) est une bibliothèque JavaScript Front permettant de créer des interfaces utilisateur. Elle est maintenue par Facebook et une communauté de développeurs individuels et d'entreprises. React est utilisée pour construire des composants d'interface utilisateur réutilisables.

VUE

Vue.JS, ou simplement Vue, est un framework progressif pour les interfaces utilisateur pour les apps et sites JavaScript. Il s’agit d’un des frameworks front-end JS les plus populaires Un développement Vue peut donc être vu comme un composant intégrant des sous-composants. Par leur approche et leur ressemblance, Vue et React partagent de nombreux points communs.

NUXT

Nuxt.js va, au travers de son auto-configuration, abstraire plusieurs concepts afin de nous fournir une base intéressante de projet. Nuxt fournit une surcouche au routeur natif de vue qui permet de générer des routes en fonction des fichiers présents dans le dossier. Le nom du fichier est directement utilisé comme nom de route et le composant présent sera utilisé comme point d’entrée.

GIT

Git c’est un outil parfait pour manager mes projets et éviter des erreurs couteuses.

FIGMA

Figma est ou outil de design collaboratif qui permet aux UX/UI designers de prototyper les interfaces graphiques. Il aide notamment à concevoir sites web, applications et autres interfaces utilisateur.

MY SQL

Langage informatique utilisé pour exploiter des bases de données. Il permet de façon générale la définition, la manipulation et le contrôle de sécurité de données.
